# Rollowa

## 프로젝트 소개
* 구디아카데미 풀스택개발자 양성과정에서 진행한 파이널 팀 프로젝트 입니다.
* 놀이공원 사내 직원들을 위한 그룹웨어 웹사이트 입니다.

## 팀원소개 및 역할분담
|팀원|역할|
|----|----|
|유가림|마이페이지, 게시판, 어트랙션, 이용권매출|
|김호관|조직관리, 구성원관리, 예약관리|
|배기웅|로그인, 마이페이지, 알림, 채팅|
|이문희|전자결재, 이용권결제|
|이예찬|부서일정, 회사일정, 휴가관리, 이용권결제|
